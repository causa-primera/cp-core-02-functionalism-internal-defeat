# Terminology (Controlled Vocabulary)

## Canonical pairs (ES → EN)
- funcionalismo → functionalism
- derrota interna → internal defeat
- equivalencia funcional → functional equivalence
- normatividad → normativity
- razón (sentido estricto) → reason (strict sense)
- suspensión genuina → genuine suspension
- compromiso / firma normativa → commitment / normative signature
- imputabilidad / respondibilidad → imputability / answerability
- externalización → externalization (delegation boundary)

## Rule
Any change to a canonical translation must be recorded in `TRANSLATION_LOG.md`.
